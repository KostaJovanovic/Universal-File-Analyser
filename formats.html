Every file type Analyser can open.

Analyser reads and identifies 1271 file formats, grouped by category below. Each is read straight from its bytes in your browser - drop a file on the home page to analyse it; nothing is uploaded. A Full badge means the format opens in a dedicated viewer with deep metadata; an ID badge means it is identified and its header metadata is read.

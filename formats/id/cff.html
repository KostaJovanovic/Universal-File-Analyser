What is a .CFF file?

.CFF is a Native code, ML & misc (more) file (Data & code). Analyser identifies a .CFF file and reads the metadata in its header, right in your browser.

Did you know
  • A CITATION.cff file is a Citation File Format file, plain-text YAML telling people exactly how to cite a piece of software or a dataset.
  • GitHub reads a repository's CITATION.cff and shows a ready-made Cite this repository button from it.
  • The format was created so academic credit for research software could be captured in a simple, machine-readable file.
What Analyser reads
Identify and read cross-platform binaries, ML models and build artefacts: ONNX neural-network models (the framework that exported it and the IR/model version), native Node.js/Electron add-ons (.node) and macOS dynamic libraries (.dylib - Mach-O/PE/ELF container and architecture), LevelDB/RocksDB tables (.ldb), Git pack reverse indexes (.rev - RIDX header), Citation File Format metadata (CITATION.cff, content-gated against binary CFF fonts), compiler response files (.rsp), Razer Chroma animations (.chroma), build stamps/markers (.stamp) and pattern/grid data (.cgp).
Depth of analysis
.CFF is an identification-grade format: Analyser recognises it from its bytes and decodes the header metadata it carries, rather than opening it in a full viewer. Formats that do get a full viewer are marked "Full" on the formats page.
Open a .CFF file
Drag a .CFF file onto the Analyser home page (or tap to pick one). It is identified entirely in your browser - nothing is uploaded, there is no account, and it works offline once installed.

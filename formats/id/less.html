What is a .LESS file?

LESS is a Less stylesheet - a CSS preprocessor language that adds variables, nesting, mixins and functions, compiled down to ordinary CSS.

Did you know
  • Less was created by Alexis Sellier in 2009 and was, for years, the styling language behind Bootstrap.
  • Less can run in the browser through a JavaScript compiler, but in production it is almost always pre-compiled to plain CSS first.
  • Its syntax is a superset of CSS, so any valid CSS file is also a valid .less file.

Depth of analysis
.LESS is an identification-grade format: Analyser recognises it from its bytes and decodes the header metadata it carries, rather than opening it in a full viewer. Formats that do get a full viewer are marked "Full" on the formats page.
